# Canada Men's National Team: 16-Year (4 WC Cycles) Performance Analysis

## Project Question
Has the Canadian Men's National Team demonstrated measurable improvement over 
the last four World Cup cycles, and what factors best explain that improvement?

## Data Sources
- International football results: [[Kaggle link](https://www.kaggle.com/datasets/martj42/international-football-results-from-1872-to-2017?resource=download)]
- FIFA World Rankings:[ [Kaggle link](https://www.kaggle.com/datasets/cashncarry/fifaworldranking/code)

## Tools
SQL Server Management Studio (data cleaning, classification, analysis views)
Power BI (visualization)

## Key Findings
Evidence suggests Canada's success reflects genuine team improvement both when facing weaker opposition 
in qualifiers and mid-high level opposition in international friendlies and tournament finals.
